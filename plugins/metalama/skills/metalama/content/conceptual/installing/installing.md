# Installing Metalama

This section explains how to install Metalama.

Metalama's core features are implemented through NuGet packages, which don't require machine-level installation. However, we recommend installing the Visual Studio Tools for Metalama when working with Metalama projects.

| Article  | Description  |
|---------|---------|
| <xref:install-vsx>    | Explains how to install the Visual Studio Tools for Metalama. We recommend this extension for all users planning to use Metalama.        |
| <xref:dotnet-tool>    | Explains how to install the Metalama Command Line Tool. This is optional during initial evaluation.        |
| <xref:register-license>     | Explains how to transition from trial mode to Metalama Free or register a license key. You can skip this while your trial period is active.        |
